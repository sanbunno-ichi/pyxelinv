# Pyxel Invaders

## 概要
- ゲーム内容はスペースインベーダー。
- なんとなくこんな感じだったかなと作っています。
- 効果音は、frenchbreadさん作成の「Pyxel RPG SE パック」を使用しています。（歩行音のみ追加）

- カラーセロハンモードと白黒があります。
- キーボード、ジョイパッド、バーチャルゲームパッドに対応しています。
- キーボードでのショットキーは「Z」キー。
